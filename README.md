# Mern Floras Companion

### Frontend Stack

```
React 18
React Router 6
Tailwind
Daisy UI
Vite
```

### Backend Stack

```
Express Js
MongoDb
Node Js
Tailwind
JWT
```

## Project Setup

### Client Side

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

#

### Server Side

```sh
npm install
```

### For this project, you will need a .env file that contains following fields;

```
PORT
MONGO_URI
SECRET
```

### Compile and Hot-Reload for Development

```sh
npm run start
```
